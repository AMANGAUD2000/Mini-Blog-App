Mini Blog App

by: Aman Gaud

This is a mini Blogging App created using Django Framework. Frontend of the application is created using HTML, CSS, and bootstrao 5. And Backend is created using django. Sqlite is used for database.

On the Top you can see the navigation bar to navigate througout the site. On the top right corner you can see an option to login. when login is clicked your are redirected to login page. If you dont have an accout you can easy create accout their by clicking on the signup link present on the login page. once you get login the you are redirected to this home page .

When you click on the Blogs Link on the navbar then you are redirected to the blogs section where you can see the list of all available blogs. To read any particular blog you can click on the blog title to read about it in detail.

When you click on the Bloggers link available on the navbar you can see the list of all the bloggers available on the platform. When you click on the Bloggers name you are redirected to a page where you can see bloggers bio. Also you can read the blogs contributed by him/her. Finally you can logout using logout button on the top right corner of the navbar.
